# Getting Started

Welcome to the **Career Knowledge Framework (CKF)**.

CKF is an evidence-based methodology for transforming human experience into structured, authoritative career knowledge. Rather than maintaining independent resumes, cover letters, LinkedIn profiles, portfolios, and job applications, CKF establishes a single authoritative knowledge source from which consistent career artifacts can be generated.

The framework is designed to improve traceability, consistency, and long-term maintenance of professional information while preserving the provenance of the underlying experiences.

This repository is **documentation-first**. Before creating artifacts or contributing to the framework, become familiar with the project's purpose, governance, and methodology.

# Repository Philosophy

The Career Knowledge Framework is **not** a resume-writing project.

It is a knowledge architecture for representing professional experience through a governed chain of authoritative and derivative artifacts.

The framework separates:

- Evidence from interpretation.
- Experience from capability.
- Capability from presentation.

This separation allows career artifacts to evolve without repeatedly rewriting the underlying history from which they are derived.

---

# Current Status

The repository is in its foundational phase.

Current work is focused on:

- Establishing governance.
- Defining artifact responsibilities.
- Standardizing methodology.
- Creating authoritative artifact specifications.

Automation, schemas, validation, tooling, and AI-assisted generation will be introduced only after the framework has stabilized.

# Contributing

Contributors are encouraged to understand the framework before proposing changes.

Governance documents should be considered authoritative. Changes to methodology or artifact responsibilities should originate through governance rather than ad hoc modifications to individual artifacts.
